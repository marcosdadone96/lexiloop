LEXICOIL — PROJECT VISION
=========================

LexiCoil must become a Universal CEFR Content Engine.


SUPPORTED LANGUAGES
-------------------
- German
- English
- Spanish


SUPPORTED PROVIDERS
-------------------
- Goethe
- Cambridge
- DELE


SUPPORTED CONTENT
-----------------
- Full Exams
- Mini Exams
- Reading
- Listening
- Writing
- Speaking
- Grammar
- Vocabulary
- Flashcards
- Stories
- Dialogues


ARCHITECTURAL PRINCIPLES
------------------------

The architecture must be CEFR-first.

Provider-specific logic must be isolated.

Knowledge must never be duplicated.

Every content generation must pass through:

  Knowledge Engine
    → Content Specification
    → Prompt Builder
    → LLM
