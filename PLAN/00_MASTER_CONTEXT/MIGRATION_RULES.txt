LEXICOIL — MIGRATION RULES
==========================

RULE 1
No duplicated knowledge.

RULE 2
CEFR definitions must exist only once.

RULE 3
Grammar definitions must exist only once per language.

RULE 4
Provider definitions must contain only:
- structure
- timing
- scoring
- task types

RULE 5
No provider-specific prompts.

RULE 6
No level-specific prompts.

RULE 7
Prompt generation must be centralized.

RULE 8
Every generated content must originate from ContentSpecification.
