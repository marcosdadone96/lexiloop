PROMPT ENGINE — RULES
=====================

Forbidden:

  buildGoethePrompt()
  buildCambridgePrompt()
  buildB1Prompt()
  buildB2Prompt()
  goethePromptFor()
  cambridgePromptFor()
  ExamModulePrompts.chunksFor() with level-specific branches

Allowed:

  buildPrompt(specification)


IMPLEMENTATION NOTES
--------------------

PromptBuilder reads ContentSpecification only.

Provider differences express as:
  - spec.examStructure.modules
  - spec.constraints.chunkPlan
  - spec.constraints.taskTypes
  - spec.outputSchema

Level differences express as:
  - spec.canDoStatements
  - spec.constraints.textLength
  - spec.grammarTopics

Language differences express as:
  - spec.locale
  - instruction language in template shell (one template, parameterized)

Chunked exams:
  buildPrompt returns { mode: 'chunks', chunks: [...] }
  each chunk prompt includes spec snapshot + single module slice
